Condition Monitoring Dashboard Interfaces

This dashboard was implemented as part of my thesis work at KIEFEL GmbH. This application is to display the interface that I worked and developed for 6 months at KIEFEL. I developed both the frontend and backend to monitor and visualize the pressing force values of thermoforming machines. The project involved preparing mock machines that send data every 30 seconds for each machine cycle, storing this data in InfluxDB, and writing Flux queries to retrieve the data. I thoroughly tested these queries using Postman API before creating services to display the data on the frontend. And also did the Database comparison.
